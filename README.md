## ขั้นตอนการติดตั้ง 
1. Clone the repository
```bash
  git clone https://github.com/Frankbanna/Deploy-Scale-MeUp.git
```
2.  Install NPM packages
    
  ```bash
    npm install
```
3. Run development mode
    
  ```bash
   npm run dev 
```
